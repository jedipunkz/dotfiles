    dotfiles for jedipunkz

    This is private dot files for @jedipunkz.

    how to use

    git clone and exec setup.pl.

    % git clone git@github.com:jedipunkz/dotfiles.git
% cd dotfiles
% perl setup.pl

    how to modify

    % emacs $HOME/.zsh # this is symlink to $HOME/dotfiles/.zsh
% cd $HOME/dotifiles
% git add .zsh
% git commit -m 'mod for .zsh'
% git push -u origin master
